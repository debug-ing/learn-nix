# Python

This is a simple Python project project with pandas,sklearn

## Prerequisites

- Nix (for managing the environment)

## Setup

1. Install Nix

    ```bash
    curl -L https://nixos.org/nix/install | sh
    ```

2. Enter the Nix shell:

    ```bash
    nix-shell
    ```

3. Build the project:

    ```bash
    python main.py 
    ```
